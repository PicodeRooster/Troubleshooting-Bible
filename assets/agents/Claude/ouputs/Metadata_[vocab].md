# Metadata

**metadata** (*noun*) — Information that describes a data point or data set, separate from the content of the data itself. Metadata includes details such as author, creation date, file size, file type, and keywords. Often summarized as "data about data," metadata improves a data system's ability to organize, search for, and retrieve information. Different types of metadata serve different purposes: descriptive metadata identifies and locates data; technical metadata describes how data is formatted and stored; administrative metadata governs access and retention; and preservation metadata ensures data remains usable over time.

*Example: For a digital photo, the metadata includes the camera model, date and time of capture, file size, and resolution — details about the image that are distinct from the image itself.*
