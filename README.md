# Troubleshooting Bible

The Troubleshooting Bible is a central reference document for IT professionals, system administrators, and support teams. It collects proven diagnostic methods, standard procedures, common error patterns, and practical fixes in one searchable master guide.

Key strengths:
- Consolidates recurring problems across hardware, software, networking, and system services
- Provides clear workflows for identifying root cause and applying fixes
- Includes shortcuts, command references, and escalation tips
- Designed for consistent onboarding and faster recovery from incidents

This living document is meant to reduce downtime, improve response quality, and keep troubleshooting knowledge accessible across teams.

Uses Obsidian to store entries as .md files and Lunar.js to add a search function. All documents link to a master index.html file.
