**Antivirus** (or antimalware) is software designed to detect, quarantine and remove malicious code from computers and mobile devices. They provide features for system administrators and individual users to secure machines from everyday cyber-attacks. 

Standard antivirus software includes features such as:

 - **Real-time protection:**  The system is actively scanning incoming files, monitoring behavior, and blocking threats before they cause damage. 

 - **Scheduled scans:** Deep scans performed weekly can uncover dormant threats that may have slipped past real-time protection.

  - **Automatic updates:** Malware evolves daily. AV software updates its threat definitions automatically to ensure real-time protection and scheduled scans remain effective.

Enterprise environments often use centralized management tools that allow administrators to monitor all endpoints, enforce policies, and receive alerts in real time..
