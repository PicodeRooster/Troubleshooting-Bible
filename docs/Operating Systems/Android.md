The dominant mobile OS, is open source and widely used on smartphones and tablets. It’s highly customizable and supports a vast app ecosystem, though device fragmentation can pose compatibility issues.

### App Installations

1. Install apps through Google Play Store or side-loading APKs (when enabling “Unknown sources”)
2. Manage permissions through Settings > Apps > [App Name] > Permissions. Includes permissions for Camera, Location, Contacts, Microphone, etc. Since Android 6.0, users can approve or deny permissions per feature.
3. Monitor usage with “Digital Wellbeing” tracks screen time and notifications

- Open-source
- Developed by Google
- Runs on devices by various manufacturer’s
- More hardware diversity
- More variability in user interface, update schedules, and pre-installed apps
- Customization with high launchers and widgets
- Offers default Google Play and 3rd-party stores
- FIle system access is semi-open via file manager
- Backup options include Google One and manual tools
- Updates vary by device and vendor
